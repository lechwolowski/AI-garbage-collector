# SZI2020Project

## Instalacja i uruchomienie

Wymagania:

```
Python: 3.7.7
```

macOS / Linux

```
make install
make start
```

or manually:

```
python -m venv env
source env/bin/activate
pip install -r requirements.txt
python main.py
```

Windows

```
python -m venv env
env\scripts\activate.bat
pip install -r requirements.txt
python main.py
```
